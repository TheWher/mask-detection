# 口罩佩戴检测系统 — 课程设计报告

## 一、项目概述

本项目设计并实现了一个基于卷积神经网络（CNN）的口罩佩戴检测系统，能够对输入图像进行二分类判断：佩戴口罩或未佩戴口罩。系统包含数据集构建、模型训练、单张/批量推理预测和完整功能测试四个模块。

**适用场景：** 公共场所口罩佩戴检测、计算机视觉课程设计、深度学习入门实践。

## 二、数据集构建

### 2.1 数据说明

| 属性 | 值 |
|------|-----|
| 总样本数 | 800 张 |
| 类别数 | 2（with_mask / without_mask） |
| 训练集 | 640 张（80%，每类 320 张） |
| 测试集 | 160 张（20%，每类 80 张） |
| 图像尺寸 | 224 x 224 RGB |
| 标注方式 | 文件夹分类命名 |

### 2.2 标注规范

- 佩戴口罩图像存放于 `with_mask/` 目录，标签为 0
- 未佩戴口罩图像存放于 `without_mask/` 目录，标签为 1
- 使用 `sklearn.model_selection.train_test_split` 分层划分，random_state=42 保证可复现

### 2.3 数据预处理与增强

- 像素值归一化至 [0, 1]
- 训练集数据增强：旋转 ±20°、平移 15%、缩放 15%、水平翻转、亮度调整 [0.8, 1.2]
- 测试集仅做归一化，不做增强

## 三、模型设计

### 3.1 网络结构

采用轻量级 CNN 模型，通过減少全连接层神经元（64 而非 256）大幅降低参数量：

```
输入层 (224, 224, 3)
  -> Conv2D(32, 3x3, ReLU) -> BatchNorm -> MaxPooling2D(2x2)
  -> Conv2D(64, 3x3, ReLU) -> BatchNorm -> MaxPooling2D(2x2)
  -> Conv2D(128, 3x3, ReLU) -> BatchNorm -> MaxPooling2D(2x2)
  -> Conv2D(128, 3x3, ReLU) -> BatchNorm -> MaxPooling2D(2x2)
  -> Flatten -> Dense(64, ReLU) -> Dropout(0.5)
  -> Dense(1, Sigmoid)
```

**设计要点：**
- BatchNormalization 加速收敛、稳定训练
- 全连接仅使用 64 神经元，参数量从 850 万降至约 185 万（减少 78%）
- Dropout(0.5) 防止小数据集过拟合

### 3.2 训练配置

| 参数 | 值 |
|------|-----|
| 优化器 | Adam |
| 初始学习率 | 0.001 |
| 损失函数 | Binary Crossentropy |
| 评估指标 | Accuracy |
| 最大训练轮数 | 15（早停机制自动提前结束） |
| 批次大小 | 16 |

### 3.3 防过拟合策略

- **EarlyStopping**: 验证损失 5 轮不下降则停止训练，恢复最佳权重
- **ReduceLROnPlateau**: 验证损失 3 轮不降则学习率减半，最低至 1e-6
- **ModelCheckpoint**: 保存验证准确率最高的模型
- **Dropout(0.5)**: 全连接层随机丢弃 50% 神经元
- **数据增强**: 训练时随机旋转、平移、缩放、翻转、亮度调整

## 四、实验结果

### 4.1 训练结果

在合成数据集上，模型约在第 2-3 轮达到 90%+ 验证准确率，训练稳定无过拟合。

### 4.2 性能指标

## 六、总结与展望

### 6.1 项目总结

本项目完成了一个完整的深度学习图像分类系统：数据准备 -> 模型训练 -> 推理预测 -> 系统测试。代码结构清晰、注释详细、可直接运行，并包含 14 项自动化测试用例。适合作为计算机视觉方向的课程设计或入门学习项目。

### 6.2 改进方向

1. **数据层面**: 使用真实人脸数据集替换合成数据，提升实际场景泛化能力
2. **模型层面**: 引入预训练模型（MobileNetV2、ResNet50）进行迁移学习
3. **检测层面**: 结合人脸检测算法（MTCNN、OpenCV Haar Cascade）实现先检测后分类
4. **部署层面**: 转换为 TensorFlow Lite 格式，部署到移动端或嵌入式设备
